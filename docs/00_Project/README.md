# Lemon Platform Project Pack

本資料夾是 Lemon Platform 的專案入口文件，提供新成員、維護者與 AI Agent 快速理解專案背景、系統範圍、Repository 結構與常用連結。

## 使用方式

建議閱讀順序如下：

1. 先讀 `PROJECT_CONTEXT.md`，了解 Lemon Platform 的背景與目標。
2. 再讀 `SYSTEM_OVERVIEW.md`，掌握平台中心、資料流與整合方向。
3. 接著讀 `REPOSITORY_MAP.md`，確認本 repo 與周邊系統的邊界。
4. 最後讀 `LINKS.md`，找到常用入口與後續待補連結。

## 維護原則

Project Pack 只放文件與規劃資料，不放程式碼、不放私密金鑰、不放一次性輸出。任何架構、流程或模組責任變更，都應同步更新本區文件。

## 文件狀態

目前文件皆為 Draft，可作為工作基準，但仍需隨實作、營運流程與系統整合結果持續修正。
