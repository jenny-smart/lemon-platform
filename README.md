# Lemon Platform

Lemon Platform 是檸檬家事的整合營運平台專案控制中心。

此 Repository 不直接承載主要業務程式碼，而是用來管理：

- 系統架構
- 專案文件
- Repository 對照
- AI 工作指派
- Roadmap
- Backlog
- 架構決策

## 目前整合範圍

- service-lemonsystem
- orders-system
- memo-system
- tool-system
- salary-system
